jdk环境肯定是作为Java学习的第一步的。
作为第一个Java博客，就来写下Mac安装jdk的过程

安装准备

  • 一台macOS电脑
  • 一个能上网的浏览器
  • 一个终端（在实用工具里面）
  • 一个脑子

安装jdk

下载openjdk

首先到华为云开源镜像站搜索「openjdk」

点击下载，会提示登录账户，我们不需要登录账户，直接点击「继续使用」

选择你需要的版本下载，记住要是OSX的版本的
（这里我选择的是11.0.1）

之后等待下载即可

配置jdk环境

将下载的包解压，会得到一个文件夹，将此文件夹放到你喜欢的目录（PS:为防止未知错误，请不要放到有空格的或者中文的文件夹名的文件夹下）
我这里就放到了根目录，这样就很少会出现问题且十分方便
